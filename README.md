# 🚀 Web Development Journey - Portfolio Esercitazioni

Benvenuti nel mio repository dedicato allo sviluppo web. Questa collezione raccoglie i progetti principali realizzati durante il mio percorso di studi, focalizzandosi sull'apprendimento di logiche di programmazione, gestione dei dati e interfacce utente dinamiche.

Ogni progetto è stato sviluppato **manualmente**, partendo da zero, per consolidare i concetti fondamentali del Web Development moderno.

---

## 🛠️ Stack Tecnologico & Competenze Acquisite

Nel corso di queste esercitazioni ho approfondito l'utilizzo delle seguenti tecnologie:

* **Frontend Core**: HTML5 semantico e CSS3 (Layout flessibili, Responsive Design).
* **JavaScript (ES6+)**: Manipolazione del DOM, gestione degli eventi e logica asincrona per il calcolo dei dati.
* **API & JSON**: Integrazione di servizi esterni (OpenWeatherMap) tramite `Fetch API` e gestione di dati geolocalizzati in tempo reale.
* **Backend & Server Side**: Sviluppo in **PHP** per l'elaborazione di dati lato server e parsing di file **XML** dinamici per la gestione di feed notizie.
* **Version Control**: Utilizzo di **Git** per il tracciamento del codice e l'organizzazione del workflow di sviluppo.
