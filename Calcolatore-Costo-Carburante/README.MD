# ⛽ Calcolatore Costo Carburante (Progetto di Verifica)

Un'applicazione web sviluppata durante una prova di verifica per calcolare il costo stimato di un viaggio in base al consumo del veicolo, alla distanza e al prezzo del carburante.

## 🚀 Caratteristiche del Progetto
- **Sviluppo Autonomo (No AI)**: Realizzato interamente a mano durante una sessione di verifica in classe, dimostrando padronanza della logica JS e del CSS.
- **Logica di Calcolo**: Implementazione della formula $\text{Costo} = (\text{Distanza} / 100) \times \text{Consumo} \times \text{Prezzo}$ con arrotondamento a due cifre decimali.
- **Validazione Dati**: Sistema di controllo integrato per impedire l'inserimento di valori nulli o negativi.
- **Interfaccia Dinamica**: Selezione del tipo di carburante tramite radio button personalizzati e layout responsive in Dark Mode.

## 🛠️ Tecnologie Utilizzate
- **HTML5**: Form di inserimento dati.
- **CSS3**: Styling avanzato per selettori e contenitori con ombreggiature.
- **JavaScript (Vanilla)**: Gestione degli eventi e calcoli matematici.

## 📂 Struttura dei File
- `index.html`: Struttura del calcolatore.
- `style.css`: Design e personalizzazione estetica.
- `index.js`: Validazione e logica di calcolo.
- `logo.jpg` / `finestra.png`: Risorse grafiche del progetto.

## 🚀 Come utilizzarlo
1. Apri `index.html`.
2. Inserisci il consumo medio della tua auto (L/100km).
3. Inserisci i km da percorrere e il prezzo attuale al litro.
4. Clicca sul tasto per ottenere il preventivo del viaggio.
