# 📰 Il Quaroit - Notiziario Dinamico (Corato Edition)

**Il Quaroit** è un aggregatore di notizie dinamico sviluppato in **PHP**. Il nome è un omaggio ironico e dialettale alla città di **Corato (BA)**, nato per trasformare un serio portale di informazione in un esperimento tecnico dal sapore locale.

---

## 🚫 PRECISAZIONE: SVILUPPO 100% MANUALE (NO AI)
Questo progetto è stato realizzato interamente **senza l'ausilio di Intelligenze Artificiali**. 
- La logica di parsing XML è stata scritta a mano.
- Il design e la struttura PHP sono frutto di studio e applicazione manuale.
- L'ironia e il concept "Il Quaroit" sono farina del sacco dello sviluppatore.


---

## 🛠️ Caratteristiche Tecniche

* **Backend PHP**: Il cuore dell'applicazione elabora i dati lato server, gestendo le richieste e la visualizzazione dinamica.
* **Repubblica XML News Parsing**: Anche se il sito "parla coratino", i dati reali (titoli, foto e link) vengono estratti direttamente dai feed ufficiali di **Repubblica.it** tramite il file `data.xml`.
* **Slider Owl Carousel**: Integrazione manuale di JQuery per la gestione dello scorrimento delle notizie.
* **Logica di Filtraggio**: Implementazione di funzioni PHP per smistare le notizie in base alle categorie e alle date.

---

## 🚀 Come avviare il progetto (Localhost)

Essendo basato su PHP, il progetto richiede un ambiente server locale:

1.  **Ambiente**: Installa **XAMPP**.
2.  **Percorso**: Copia la cartella del progetto in `C:\xampp\htdocs\Notizie`.
3.  **Attivazione**: Avvia il modulo **Apache** dal pannello di controllo XAMPP.
4.  **Browser**: Digita l'indirizzo: `http://localhost/Notizie/index.php`.

---

## 📂 Struttura del Repository

* `index.php`: Logica di backend e scheletro della pagina.
* `data.xml`: Database XML con le notizie aggiornate (Feed Repubblica).
* `style.css`: Styling originale e responsive.
* `index.js`: Gestione della sidebar e interazioni client-side.
* `assets/`: Loghi e icone grafiche del brand "Il Quaroit".
